# TECH STACK CONTEXT
Project Name: pmb-s3-stie-mahardhika (Vite React TS)

## Core Technologies
- **React 19 & Vite**: Use latest React 19 features (e.g., improved 'use' hook, ref as props).
- **TypeScript**: Strict mode enabled.
- **Tailwind CSS v4**: Use the new v4 syntax. No need for heavy config files if using CSS-first approach.

## Key Libraries
- **Routing**: `react-router-dom` v7.
- **State & Data Fetching**: `@tanstack/react-query` v5.
- **Form Handling**: `react-hook-form`.
- **UI & Animation**: `framer-motion`, `lucide-react`, `radix-ui` components.
- **Authentication**: `better-auth`.
- **Internationalization**: `i18next` with `react-i18next`.
- **Utilities**: `clsx`, `tailwind-merge` (use `cn()` utility).
